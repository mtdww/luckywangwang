## 3. 常用命令

```shell
# 拉取 Neo4j 镜像
docker pull neo4j

# 查看已拉取的镜像：
docker images | grep neo4j

# 运行 Neo4j 容器
docker run -d \
  --name my-neo4j \
  -p 7474:7474 \
  -p 7687:7687 \
  -e NEO4J_AUTH=neo4j/test123 \
  neo4j:latest
```
