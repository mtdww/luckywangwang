> OS: Ubuntu 24.04

1. tmux 安装

```bash
sudo apt update
sudo apt install tmux
```

2. 验证安装

```bash
tmux -V
```

3. 基本使用

```bash
启动新会话：tmux

列出会话：tmux ls

附加到会话：tmux attach -t <会话名>

退出会话：Ctrl+b d
```
